# Process & Resource Manager

## How to Run the Code

### 1. Open Terminal

Navigate to the project folder:

```bash
cd path/to/project/src
```

### 2. Compile the Java Files(if not compiled)

```bash
javac *.java
```

### 3. Run the Program

Run with an Input File:

```bash
java main input.txt
```

Run with Terminal Input:

```bash
java main
```
